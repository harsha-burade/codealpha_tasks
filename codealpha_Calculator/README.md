# 🧮 Basic Calculator Web App

A simple and responsive **Calculator Application** built using **HTML, CSS, and JavaScript**.
It performs basic arithmetic operations and supports both button clicks and keyboard input.

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧼 Clear screen (C button)
* ⌨️ Keyboard support
* ⚡ Real-time calculation
* 🎨 Smooth UI with hover effects

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Project Structure

calculator-app/
│── index.html
│── README.md

## 🎯 How It Works

* Click buttons or use keyboard input to enter values
* Press `=` or Enter to calculate result
* Press `C` or `c` to clear screen
* Backspace removes last digit
