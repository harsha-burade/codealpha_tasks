# 🎵 Music Player Web App

A modern and responsive **Music Player** built using **HTML, CSS, and JavaScript**.
It allows users to play songs, manage playlists, and control audio with an interactive UI.

---

## 🚀 Features

* ▶️ Play / Pause functionality
* ⏭️ Next & Previous track controls
* 🖼️ Album cover display
* 📜 Playlist support (click to play any song)
* 🔀 Shuffle mode (random playback)
* 🔁 Repeat mode (loop current song)
* ⏱️ Progress bar with real-time updates
* 🔊 Volume control slider
* 📱 Mobile-friendly UI design

---

## 🛠️ Technologies Used

* HTML5 (Structure)
* CSS3 (Styling & Responsive Design)
* JavaScript (Audio Control & Logic)

## 🎯 How It Works

* Songs are stored in a JavaScript array
* Clicking a song loads and plays it
* Controls allow navigation between songs
* Progress bar updates as the song plays
* Shuffle mode plays random songs
* Repeat mode loops the current song
