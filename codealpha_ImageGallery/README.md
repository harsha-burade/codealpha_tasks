# 📸 Responsive Image Gallery

A modern and fully responsive **Image Gallery Web App** built using **HTML, CSS, and JavaScript**.
It includes features like lightbox preview, image navigation, filters, and image upload.

## 🚀 Features

* 📷 Display 20+ images dynamically
* 🔍 Lightbox view for full-size image preview
* ⏭️ Next / Previous navigation
* 🎨 Hover effects with smooth transitions
* 📱 Fully responsive design (works on mobile, tablet, desktop)
* 🗂️ Filter images by category (Nature / City)
* ⬆️ Upload your own images dynamically

## 🛠️ Technologies Used

* HTML5
* CSS3 (Grid + Responsive Design)
* JavaScript (DOM Manipulation)

## 📂 Project Structure

📁 image-gallery
 ├── index.html
 ├── style.css

## 🎯 How It Works

* Images are generated dynamically using JavaScript.
* Clicking an image opens it in a lightbox view.
* Users can navigate using Next/Prev buttons.
* Filters allow sorting images by category.
* Users can upload images which are instantly added to the gallery.
